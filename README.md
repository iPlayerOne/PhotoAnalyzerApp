# PhotoAnalyzerApp

iOS-приложение на SwiftUI, выполненное в рамках тестового задания: импорт фотографий, распознавание лиц через Vision, локальное сохранение проектов и экспорт изображения.

## Задание

Разработать приложение, которое позволяет пользователю выбрать фотографию из галереи, определить наличие лица на изображении с помощью нативного фреймворка Vision и сохранить результат локально.

Также должна быть реализована возможность экспорта изображения в файловую систему устройства.

## Описание

Приложение содержит основной сценарий работы с фото:

- выбор изображения из галереи через PhotosPicker;
- отображение overlay-экрана обработки с затемнением, blur-эффектом, skeleton-состоянием и индикатором загрузки;
- автоматическая генерация имени проекта;
- распознавание лица на изображении через Vision;
- сохранение изображения в файловое хранилище приложения;
- сохранение метаданных проекта в SwiftData;
- отображение сохраненных проектов в списке;
- открытие сохраненного проекта в overlay;
- экспорт сохраненного изображения через системный Share Sheet.

Для локального хранения метаданных используется SwiftData. Оригинальные изображения сохраняются отдельными файлами в Documents-директории приложения.

## Архитектура

Проект разделен на несколько слоев:

- `App` - точка входа, RootView и контейнер зависимостей;
- `Core/Models` - общие модели приложения;
- `Core/Persistence` - SwiftData entity и repository для проектов;
- `Core/Services` - генерация имен, Vision-детекция лица, файловое хранилище, определение ориентации и подготовка превью;
- `Core/UI` - UI-инфраструктура и Share Sheet;
- `Features/Projects` - основной экран проектов, карточки, overlay обработки и ViewModel.

## Возможности

- Импорт одного изображения из галереи.
- Запуск PhotosPicker из empty state и через кнопку в верхней панели.
- Генерация имени проекта из заданного набора слов с индексом.
- Распознавание лица на изображении через Vision.
- Отображение skeleton-состояния во время обработки.
- Отображение результата обработки с иконкой лица или изображения.
- Локальное сохранение проекта через SwiftData.
- Сохранение исходного изображения в Documents-директорию приложения.
- Список сохраненных проектов.
- Разная высота карточек в зависимости от ориентации изображения.
- Открытие сохраненного проекта из списка.
- Экспорт изображения через системный Share Sheet.

## Библиотеки

Сторонние библиотеки не использовались.

В проекте используются только нативные фреймворки Apple:

- SwiftUI - построение интерфейса;
- SwiftData - локальное хранение метаданных проектов;
- Vision - распознавание лиц на изображении;
- PhotosUI - выбор изображения из галереи;
- ImageIO - чтение метаданных изображения и создание downsampled-превью;
- UIKit - интеграция системного Share Sheet и работа с UIImage.
